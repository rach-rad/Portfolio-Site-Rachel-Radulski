Context

As part of a fast-paced project, I undertook the redesign of the NGO Culture Routes Society's app, Via Eurasia.

Problem

Via Eurasia's existing MVP had shown potential, but user feedback indicated navigational challenges, excessive screens, and outdated branding.

Objective

Create a seamless, intuitive, and visually pleasing user experience by enhancement of navigation and elevation of the visual appeal of the app while adhering to accessibility guidelines.

Solution

I meticulously examined the app's UI elements, identified areas for enhancement, and crafted an improved design. I restructured the "Home" page layout, created a new impactful "Welcome" page, and reimagined navigation for a more user-friendly interface. Accessibility was prioritized, optimizing text hierarchy and brand color usage.

Style Guide Updates

Meticulous updates were made to the color palette, logo design, and essential UI components. Through these enhancements, the app's overall user experience was not only elevated but also aligned with modern design trends and accessibility standards.

Before and After Designs

Landing Page

Annotations
  • Updated to look more modern and welcoming
  • Per the art director, legal copy and sponsors moved to about page

Route Page

Annotations
  • New logo with updated typography to support a more modern design
  • Logo monochromatic to support various backgrounds
  • The previous formatting of the cards was confusing, addition of location button at top to decrease clutter in the card

Route Detail Page

Annotations
  • Removal of top nav bar when not on a main screen
  • Rounded corners added to map container to adhere with styling of images and buttons
  • Orange map button replaces “Show on Map”
  • Description was very lengthy on original design. Drop down menu added to save space, giving the user the option of reading more information about the route

Points of Interest Page

Annotations
  • Addition of button icons to display function while keeping screen minimalistic
  • Icons updated to support outline styling used in previous screens
  • Increased padding between icons to make more easily tappable

'Check Before You Go' Page

Annotations
  • Per art director, “Check Before You Go” screen was to become a menu.
  • Updated colors to support color scheme and establish hierarchy

About Page

Annotations
  • Updated image due to poor quality of original map image.
  • Although this map does not support information, until there is an update image this will serve as a placeholder
  • Per creative director, About screen should be reorganized for visual appeal.
  • Addition of more vibrant images and simple graphics to increase engagement.
  • Per art director, legal and sponsors information should be added to this screen
  • Due to the high amount of content assigned, a menu is added to allow the user to more quickly find a section of information
